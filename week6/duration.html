CSS Keyframe Animations

Duration

The property name "animation-duration" is placed within the element's selector and defines how long the animation will take to complete. This property must be specified or no animation will begin because the default value for it is 0s, meaning zero seconds.

CSS Syntax


 @keyframes animationname {
 keyframes-selector {css-styles;}
 keyframes-selector {css-styles;}
  }

Example


 @keyframes example {
  0% {background-color: red;}
  25% {background-color: yellow;}
  50% {background-color: blue;}
  100% {background-color: green;}
  }


 element {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: example;
  animation-duration: 10s;
  }

In this example we are changing the background color from a red (at 0%) to yellow (at 25%) to a blue (at 50%) to green (at 75%) and then back to red and started with (at 100%)
