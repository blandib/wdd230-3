CSS Keyframe Animations

Multiple Styles

You can add as many keyframe selectors as needed within one animation.To change multiple styles in the same keyframes-selector (or percent) add more styles within the curly brackets after the selector.

CSS Syntax


 @keyframes animationname {
  0% {css-styles; css-styles; css-styles;}
  100% {css-styles; css-styles; css-styles;}
 }

Example


 @keyframes mymove {
  0% { top: 0px; left: 0px; background: red; }
  25% { top: 0px; left: 100px; background: blue; }
  50% { top: 100px; left: 100px; background: yellow; }
  75% { top: 100px; left: 0px; background: green; }
  100% { top: 0px; left: 0px; background: red; }
  }

In this example The background color is change everytime the that the object is moving
