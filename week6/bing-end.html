CSS Keyframe Animations

Beginning and Ending

You can add as many style changes as you like by specifying percentages. First, define your keyframes using the @keyframes rule. The percentage values represent the bing and the end during of the animation sequence when a keyframe occurs. For example: 0% indicates the initial state of the animation. 100% represents the final state of the animation.

CSS Syntax


  @keyframes example {
  0% { @keyframes; css-styles; css-styles; }
  25% { @keyframe: css-styles; css-styles; }
  50% { @keyframes: css-styles; css-styles; }
  100% { @keyframes: css-styles; css-styles;}
  }

Example


  @keyframes Example {
  0% { background-color: red; }
  25% { background-color: yellow; }
  50% { background-color: blue; }
  100% { background-color: green; }
  }

In this example, the background color of an element will change when the animation is 25%, 50%, and 100% complete.When an animation is finished, it goes back to its original style.
