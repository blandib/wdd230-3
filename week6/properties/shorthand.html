CSS Keyframe Animations

Shorthand

The shorthand animation property is a convenient way to apply an animation between styles without having to write all the individual properties.

Example


 div {
  animation-name: example;
 animation-timing-function: ease-in;
 animation-delay: 2s;
 animation-duration: 4s;
 animation-iteration-count: infinite;
 animation-fill-mode: none;
 animation: short 5s infinite;
 animation-direction: alternate;
  }

Example


 div {
 animation: 4s linear 0s infinite alternate sun-rise;
  }

This example uses all of the values specified above.
