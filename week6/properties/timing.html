CSS Keyframe Animations

Timing

The animation-timing-function specifies the speed curve of an animation.You can specify the timing of the animation by using the animation-timing-function property.which defines the time it takes for an animation to change from one set of CSS styles to another. Letâ€™s explore the different options:

Linear: The animation has the same speed from start to end.

Ease: This is the default value. The animation has a slow start, speeds up in the middle, and slows down at the end.

Ease-in: The animation has a slow start.

Ease-out: The animation has a slow end.

Ease-in-out: The animation has both a slow start and a slow end.

Cubic Bezier: You can define your own values using the cubic-bezier function with four numeric parameters (ranging from 0 to 1).

Example


  #div1 { animation-timing-function: linear; }
  #div2 { animation-timing-function: ease; }
  #div3 { animation-timing-function: ease-in; }
  #div4 { animation-timing-function: ease-out; }
  #div5 { animation-timing-function: ease-in-out; }
  #div6 { animation-timing-function: cubic-bezier; }
