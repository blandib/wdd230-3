CSS Keyframe Animations

What is a CSS Keyframe Animation?

It allow us to creating dynamic and interactive effects on web pages using CSS. It allows us to specify different styles for an element at different points in the animation. For example Make an element move gradually 200px down

CSS Syntax

@keyframes animationname{
  {keyframes-selector {css-styles;}}
}

Example

@keyframes mymove {
 from {top: 0px;}
 to {top: 200px;}
}

In this Example the element is moving to one Directions to another.
